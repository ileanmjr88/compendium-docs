## How to use a specific version

Set the version exactly in your `compendium.toml`:

```toml
[languages]
go = "1.26.1"

[tools]
cmake = "4.3.1"
```

Then run `compendium install` — the registry resolves the platform/arch
tarball for your machine, downloads it, and verifies the SHA-256 checksum.

If a version isn't listed here, it isn't installable. Open a PR against
[compendium-registry](https://github.com/ileanmjr88/compendium-registry)
to add it.
